## Overview

Apostalica Christatva is a content-driven mobile app that fetches articles, topics, announcements, and dynamic pages from a remote backend.  
It integrates with a custom **PHP + MySQL API** and offers a smooth, reliable reading experience optimized for Android devices.

The UI is fully designed in **Figma** and implemented manually with Flutter widgets for a polished, consistent interface.

---

## Key Features

### 1. **Article Delivery System**
- Fetches articles from a remote MySQL database via PHP APIs  
- Supports:
  - Full article pages  
  - Dynamic intro pages  
  - Menu-based theological content  
- Fast loading with optimized response handling  

### 2. **Topic System**
- Multi-layer topic browsing  
- Articles grouped under theological/textual categories  
- Dynamic navigation with filtered views  

### 3. **Push Notifications (FCM)**
- Firebase Cloud Messaging integration  
- Handles foreground, background, and terminated-state notifications  
- Supports deep linking (optional implementation)  

### 4. **Announcement System**
- Real-time announcement fetching from database  
- Admin-controlled broadcast messages  
- Updates appear instantly without app updates  

### 5. **Beautiful Custom UI**
- Entire layout designed in Figma  
- Pixel-perfect implementation in Flutter  
- Optimized spacing, typography, and layout for mobile devices  

### 6. **Proven Play Store Readiness**
- Successfully passed Google Play Store closed testing  
- Meets performance, safety, and stability requirements  
- **Scheduled release:** January 1, 2026
